# Seasonal Japan 🌸
A data engineering & analytics project exploring Japan's seasonal patterns — cherry blossom bloom timing, autumn foliage timing & climate patterns (temperature, precipitation, snowfall).

## Stack
- Python — ingestion, cleaning & feature engineering
- SQL — analytics models
- Local warehourse — DuckDB or PostgreSQL
- Visualization — Plotly

## Repo
- `src/` — Python pipeline code
- `sql/` — SQL transformations & models
- `data/` — raw, interim & curated datasets
- `dashboards/` — app or dashboard outputs
- `docs/` — diagrams & documentation
